# Summerfield Waldorf — Kindergarten & TK (Santa Rosa)

Responsive marketing page for Summerfield Waldorf's Kindergarten / TK program,
implemented from the Figma design system (Poppins + Mulish, olive-green brand).

- `index.html` — the page (self-contained CSS/JS)
- `images/` — photography

Open `index.html` in a browser, or serve the folder with any static host.
